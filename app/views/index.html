Repository Structure

app/The Root Directory
app/assetsCSS, Javascript and other static (unchanging) assets
app/dataData objects used by the project
app/modelsModel Objects that represent elements of the System (Stations, Trips)
app/viewsViews - A view represents a window into the data
app/testUnit Tests - These test specific units of functionality such as loading files
